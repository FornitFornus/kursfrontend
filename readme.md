# Wprowadzenie

## Pierwsze odpalenie VSC + GIT + GITHUB

1. Sprawdź, czy GIT jest zainstalowany.
   - Otwórz consolę (CTRL + ~)
   - wpisz "git --version"
2. "Zgłoś się" w GIT - wpisz w terminalu:
   - git config --global user.name "NAZWA USERA"
   - git config --global user.email "EMAIL USERA"
